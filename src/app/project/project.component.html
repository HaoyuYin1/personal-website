GA Project -- Robot Robi

Robi is a self-directed robot and its task is to pick up the garbage in a M*M map. In this game Robi have a limited steps of movements. In the limited movements Robi need to get a grade as high as possible.

Genetic Algorithm is used to find the strategies for Robot Robi.

GoPats Budget Tracker

GoPats budget tracker is a a single-page web application with REST APIs to view multiply personal credit cards information, including detailed transactions and balance, create and manage monthly categorized budgets.

Front-end framework is Angular. Back-end framework is Express, based on Node.js, together with MongoDB as the database software.
